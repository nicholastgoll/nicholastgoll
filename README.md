## Featured Project
 
### [Touchless Pong](https://github.com/nicholastgoll/touchless-pong)
 
A Pong game with a modular architecture designed around swappable input controllers, a web-based configuration layer, and signal processing for real-time input.
 
- **Built with:** Python, Pygame, Bottle, pigpio, gpiozero
- **Highlights:**
  - Strategy pattern architecture enabling seamless controller swapping at runtime
  - Lightweight Bottle web server for configuration and game settings
  - Exponential moving average (EMA) smoothing for noisy input data
  - Clean package structure with separated concerns across controllers, game logic, and UI
